Performs max pooling on the input and outputs both max values and indices.
### Aliases:
- `tf.compat.v2.nn.max_pool_with_argmax`

```
 tf.nn.max_pool_with_argmax(
    input,
    ksize,
    strides,
    padding,
    data_format='NHWC',
    output_dtype=tf.dtypes.int64,
    include_batch_in_index=False,
    name=None
)
```
