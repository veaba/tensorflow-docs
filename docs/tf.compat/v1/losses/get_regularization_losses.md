Gets the list of regularization losses.

    
    
    tf.compat.v1.losses.get_regularization_losses(scope=None)
    

#### Args:

  * **`scope`** : An optional scope name for filtering the losses to return.

#### Returns:

A list of regularization losses as Tensors.
