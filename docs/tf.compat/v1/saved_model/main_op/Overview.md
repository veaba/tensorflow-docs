SavedModel main op.

Builds a main op that defines the sequence of ops to be run as part of the
SavedModel load/restore operations.

## Functions

[`main_op(...)`](https://tensorflow.google.cn/api_docs/python/tf/compat/v1/saved_model/main_op/main_op):
Returns a main op to init variables and tables. (deprecated)

[`main_op_with_restore(...)`](https://tensorflow.google.cn/api_docs/python/tf/compat/v1/saved_model/main_op_with_restore):
Returns a main op to init variables, tables and restore the graph.
(deprecated)
