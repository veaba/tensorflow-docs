```
 tf.keras.preprocessing.image.apply_channel_shift(
    x,
    intensity,
    channel_axis=0
)
 
```

# Arguments


```
x：输入张量。必须是三维的。
强度：转化强度。
channel_axis: Index of axis for channels in the input tensor.
 
```

```
纽米图像张量。
```
