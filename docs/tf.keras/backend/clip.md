Element-wise value clipping.
### Aliases:
- `tf.compat.v1.keras.backend.clip`
- `tf.compat.v2.keras.backend.clip`

```
 tf.keras.backend.clip(
    x,
    min_value,
    max_value
)
```
#### Arguments:
- `x`: Tensor or variable.
- `min_value`: Python float or integer.
- `max_value`: Python float or integer.
#### Returns:
A tensor.
