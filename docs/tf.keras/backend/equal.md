Element-wise equality between two tensors.

**别名** : [ `tf.compat.v1.keras.backend.equal` ](/api_docs/python/tf/keras/backend/equal), [ `tf.compat.v2.keras.backend.equal` ](/api_docs/python/tf/keras/backend/equal)

```
 tf.keras.backend.equal(
    x,
    y
)
 
```

#### 参数：
- **`x`** : Tensor or variable.
- **`y`** : Tensor or variable.


#### 返回：
布尔张量
