Returns the truth value of x OR y element-wise.

**Aliases** : [ `tf.compat.v1.logical_or` ](/api_docs/python/tf/math/logical_or), [ `tf.compat.v1.math.logical_or` ](/api_docs/python/tf/math/logical_or), [ `tf.compat.v2.logical_or` ](/api_docs/python/tf/math/logical_or), [ `tf.compat.v2.math.logical_or` ](/api_docs/python/tf/math/logical_or), [ `tf.logical_or` ](/api_docs/python/tf/math/logical_or)

```
 tf.math.logical_or(    x,    y,    name=None) 
```

*NOTE*: [ `math.logical_or` ](https://tensorflow.google.cn/api_docs/python/tf/math/logical_or) supports broadcasting. More about broadcasting[here](http://docs.scipy.org/doc/numpy/user/basics.broadcasting.html)

#### Args:
- **`x`** : A  `Tensor`  of type  `bool` .
- **`y`** : A  `Tensor`  of type  `bool` .
- **`name`** : A name for the operation (optional).


#### Returns:
A  `Tensor`  of type  `bool` .
